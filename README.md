# **how-to-learn**

## О проекте:
Учебный проект, представляет собой верстку одностраничного сайта базового уровня.
Стек: HTML, CSS

## Используемые инструменты:
- Flexbox
- Grid
- Animation
- Transform
- Transition
- @Keyframes
- @Import

## Инструкции по запуску:
- Скачать или склонировать репозиторий
- Открыть `.html` файл
